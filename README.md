Приложение для покупки билетов на междугородние автобусные рейсы.

## Описание

"Автовокзал" - это iOS-приложение, разработанное на Swift, которое позволяет пользователям искать, просматривать и покупать билеты на автобусные маршруты между городами. Приложение интегрируется с сервисом easyway24.ru для обработки заказов и управления билетами.

### Основные возможности:

- Выбор пункта отправления и назначения
- Поиск маршрутов по дате
- Просмотр доступных рейсов с расписанием и ценами
- Покупка билетов через интегрированный веб-интерфейс
- Управление купленными билетами в личном кабинете
- Поддержка русского и английского языков

## Требования

- iOS 10.0 или выше
- Xcode 11 или выше
- CocoaPods для управления зависимостями

## Установка

1. Клонируйте репозиторий:

   ```
   git clone <url-репозитория>
   cd busapp
   ```

2. Установите зависимости через CocoaPods:

   ```
   pod install
   ```

3. Откройте проект в Xcode:

   ```
   open busapp.xcworkspace
   ```

4. Соберите и запустите приложение на симуляторе или устройстве.

## Зависимости

- **Alamofire** (~> 4.5): Для сетевых запросов
- **Alamofire-Synchronous** (~> 4.0): Синхронные запросы на базе Alamofire
- **FSCalendar**: Библиотека календаря
- **VACalendar**: Дополнительная библиотека календаря

## Структура проекта

- `busapp/`: Основной код приложения
  - `ViewControllers/`: Контроллеры представлений (FromViewController, ToViewController, RoutesViewController и др.)
  - `Assets.xcassets/`: Ресурсы приложения (иконки, изображения)
  - `Base.lproj/`: Основные storyboard файлы
  - `en.lproj/`, `ru.lproj/`: Локализации
- `Pods/`: Зависимости, управляемые CocoaPods

## Разработка

Проект использует стандартную архитектуру iOS с использованием Storyboard для интерфейса. Для сетевых запросов используется Alamofire. Календарь реализован с помощью FSCalendar и VACalendar.
